# Desafio Python
 
Projeto é uma avaliação para uma vaga na empresa Supera Inovação Tecnologia.

## Visão geral

O teste consiste em construir uma API HTTP de um pseudo e-commerce de games utilizando Python (preferencialmente o framework Django), e construir a interface do e-commerce utilizando React Web.

## Meu processo

- Pesquisa no Google, na documentação do Django e no youtube já que nunca de tinha trabalhado com Django.
- Os arquivos principais desse projeto estão no formato RAR.
